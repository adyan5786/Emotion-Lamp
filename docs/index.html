One-Click ESP32 Installer

Install the latest Emotion Lamp firmware directly from your browser and start using the audio-reactive web interface in minutes.

Install Firmware

Connect your ESP32 through USB, then click Install. No extra files are required from the user side.

Quick Setup

  1. Flash completes and board reboots.
  2. Connect to Wi-Fi AP Emotion-Lamp.
  3. Open http://4.3.2.1.
  4. Configure Home Wi-Fi in Wi-Fi Configurator.

What You Get

Audio-Reactive Lighting RMS and frequency-driven color and brightness behavior optimized for ESP32.
Web Control Dashboard Power, brightness, profile editing, animation selection, and tuning from one interface.
Network Recovery Built-In AP fallback and staged Wi-Fi config with safe apply/reboot workflow.

Support Notes

  1. Use a data USB cable (not charge-only).
  2. Close other serial tools before flashing.
  3. This installer flashes bootloader, partitions, and app firmware.
